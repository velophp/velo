# Velo Documentation

Welcome to the Velo documentation. Velo is a flexible, dynamic backend framework built on top of Laravel, designed to handle dynamic schemas and real-time data with ease.

## Getting Started

- **[Core Concepts](core-concepts.md)**: Learn about Velo's architecture, Collections (Base and Auth), and EAV-based schema.
- **[Authentication](authentication.md)**: Understand how Velo handles stateful authentication and multi-collection user providers.
- **[Realtime](realtime.md)**: Explore how to subscribe to data changes via WebSockets using filters.
- **[Hooks System](hooks.md)**: Learn how to extend Velo's core logic with custom hooks and triggers.

## Key Features

- **Dynamic Schema**: Define your data structure at runtime without migrations.
- **Stateful Auth**: Secure, flexible user management with session control.
- **Realtime Updates**: Push data changes to clients instantly with granular filtering.
- **Extensible Hooks**: Intercept system events to add custom business logic.
